 CSongChordsRecognizer.Graphs.FilterNthHarmonicsThis algorithm filter spectrogram from nth harmonics which are generated by instruments. That means that if the tone of frequency F is played, then it is also reflected on frequencies 'F*n' for n = 1,2,3,.... This filtration takes the minimum of spectrogram[i][j * n]* (1/epsilon)^{n-1} for n = 1,2,3,...,n_harmonics. By default epsilon=0.4 and n_harmonics=10.
