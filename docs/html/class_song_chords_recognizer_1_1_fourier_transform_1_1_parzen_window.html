Detailed Description

window function in shape of narrow hill. (n shuffled to left by N/2, N is length of window (or also 'g' part)) for |n| between 0 and (N-1)/4: w[n] = 1 - 6 * (|n|/(N/2))^{2} + 6 * (|n|/(N/2))^{3} for |n| between (N-1)/4 and (N-1)/2: w[n] = 2 * (1 - (|n|/(N/2))^{3}
