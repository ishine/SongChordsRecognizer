Detailed Description

Window function, in shape of narrow hill, simular to Parzen window. w[n] = a0 - a1 * cos(2*pi*n/N) + a2 * cos(4*pi*n/N) - a3 * cos(6*pi*n/N), where N is length of window (or also 'g' part)
